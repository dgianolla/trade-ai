# API de Análise de Imagens

API unificada em Python para processar e analisar imagens utilizando técnicas de OCR e visão computacional.
Possui dois módulos principais:
1. **Plantas**: Processamento de plantas de lojas (layouts/mapas)
2. **Análise de Fotos**: Análise geral de imagens

## Stack
- Python 3.11+, FastAPI, PostgreSQL, Redis, Celery, MinIO, Docker

## Setup
1. Copie o `.env.example` para `.env`
2. Configure as variáveis de ambiente
3. Execute com `docker-compose up --build`
